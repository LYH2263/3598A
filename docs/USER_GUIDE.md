# 使用文档

## 1. 注册与登录
1. 打开登录页，选择“去注册”。
2. 填写角色、用户名、学号（学生必填）、手机号/邮箱、安全问题与答案、密码。
3. 登录支持用户名/学号/手机号/邮箱，支持“记住我”。

## 2. 密码重置（安全问题 + 邮箱验证码）
1. 输入账号与安全问题答案，完成验证码后点击“发送邮箱验证码”。
2. 系统返回邮箱提示与演示验证码。
3. 输入邮箱验证码与新密码，提交完成重置。

## 3. 学生端功能
- 充值订单：提交金额、渠道与备注，等待管理员审核。
- 消费统计：按时间/类别筛选，查看记录表和可视化趋势。
- 余额日志：查看每次余额变化（充值、消费、冻结、解冻）。
- 公告通知：查看系统公告、订单通知与安全提醒。

## 4. 管理员端功能
- 订单审核：对学生充值单进行通过或驳回，写入审核备注。
- 用户管理：检索用户、调整角色、启用/禁用账号。
- 钱包管理：冻结/解冻用户钱包。
- 公告发布：发布公告并向全体用户推送通知。

## 5. 资金与日志规则
- 充值单仅在“审核通过”后入账。
- 消费扣费与充值入账都写入余额变动日志。
- 钱包冻结后，禁止提交充值订单与消费扣费。

## 6. FAQ
- Q: 为什么提交订单后余额未变？
  A: 充值单需管理员审核通过才会入账。
- Q: 为什么被冻结后不能操作？
  A: 系统会拦截交易操作，需管理员解冻。
- Q: 通知太多如何处理？
  A: 在通知面板点击“全部标记已读”。
